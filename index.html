Projects

University projects
  • Watertracker

    Watertracker is an Android which tracks the consumption of water of a user. The user can add custom sizes of their favorite container. It also shows the consumption over the last 7 days in a neat bar chart. And if the user forgets to drink, the app reminds to drink!

  • Robo-Project

    First year uni project using a Zumo Robot with various sensors. The robot had to follow lines, fight bravely in a sumo fight and make sure it can manage through a maze. We named the robot "Gonk".

Hobby projects
  • CircularRoads

    CircularRoads is a Unity project which achieves procedually generated full circle roads using the magic of polar coordinates and cubic interpolation. First it creates random points, then overlays splines and finish by creating a seamless mesh.
